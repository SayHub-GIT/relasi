RELASI STORE - PUBLIC CATALOG + ADMIN LOGIN ONLY
================================================

FINAL ARCHITECTURE:
===================

1. PUBLIC CATALOG (/store)
   - No authentication required
   - Users can browse laptops freely
   - Search & filter functionality
   - View-only access

2. ADMIN DASHBOARD (/admin)
   - Firebase authentication required
   - Credentials: admin@relasi.com / admin123
   - Full CRUD operations for products
   - Protected by middleware + role verification

3. HOME PAGE (/)
   - Shows "Admin Login" button
   - Shows "View Catalog" button
   - Auto-redirects if already logged in

AUTHENTICATION FLOW:
====================

Firebase (Login)
     ↓
Verify email & password
     ↓
Check admin role in Supabase
     ↓
Set firebaseUid cookie
     ↓
Redirect to /admin
     ↓
Middleware validates token for /admin routes

DATABASE:
=========
- Supabase PostgreSQL
- Tables: products, users (with roles)
- No authentication data stored (Firebase handles it)
