=== RELASI STORE - SETUP INSTRUCTIONS ===

Untuk menyelesaikan setup, lakukan langkah-langkah berikut:

1. JALANKAN DATABASE SCRIPT:
   - Buka Supabase Dashboard > SQL Editor
   - Copy semua kode dari scripts/01-init-database.sql
   - Paste dan jalankan (RUN)
   - Tunggu sampai selesai

2. SET ADMIN USER (Manual):
   - Buka Supabase Dashboard > SQL Editor
   - Jalankan query berikut setelah membuat akun:
   
   UPDATE users SET role = 'admin' WHERE email = 'your-admin-email@example.com';

3. VERCEL DEPLOYMENT:
   - Add environment variables di Vercel project settings:
     * NEXT_PUBLIC_SUPABASE_URL
     * NEXT_PUBLIC_SUPABASE_ANON_KEY

4. TEST APPLICATION:
   - Buat akun user baru
   - Login dan lihat /store (catalog view-only)
   - Ubah role di database ke 'admin'
   - Logout dan login ulang
   - Akses /admin (dashboard dengan CRUD)
